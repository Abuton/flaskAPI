# Bank Flask Route

For financial instituition Employees to perform some basic financial instituition operations like:

- edit customer
- add customer
- add account
- delete/deactivate customer
- delete/deactivate account
- transfer amount
- retrieve balance
- retrieve history (transfer/ deposit etc)
